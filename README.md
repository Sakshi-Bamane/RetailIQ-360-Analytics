# 🛒 RetailIQ 360: End-to-End Retail Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end Retail Sales Analytics solution developed using **MySQL** and **Power BI**.

The goal of this project is to analyze sales performance, customer behavior, product profitability, and shipping efficiency to generate actionable business insights and support data-driven decision-making.

---

## 🎯 Problem Statement

Retail businesses generate large volumes of transactional data every day. Without proper analysis, it becomes difficult to identify:

* Top-performing products
* High-value customers
* Sales trends
* Loss-making products
* Shipping performance

This project helps transform raw retail data into meaningful business insights.

---

## 🎯 Project Objectives

✅ Analyze sales and profit performance

✅ Identify top customers and products

✅ Detect loss-making products

✅ Monitor monthly sales trends

✅ Evaluate shipping performance

✅ Build an interactive dashboard for business users

## 🗂️ Dataset Information

The project consists of four datasets:

### 👥 Customers

Customer details and location information.

### 📦 Products

Product information and categories.

### 🧾 Orders

Sales transactions including revenue and profit.

### 🚚 Shipping

Shipping details and delivery information.

---

## 🏗️ Database Schema

The following tables were created in MySQL:

* customers
* products
* orders
* shipping

Relationships were established for analysis and reporting.
